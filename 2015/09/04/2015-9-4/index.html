如何用hexo搭建博客并且绑定域名

  • hexo
  • 博客

##环境

windows 8.1 64 bit
文本编辑器使用的是微软的Visual Code（推荐一下）

##安装hexo前的准备

###Git

首先安装Git，并配置好SSH，如果你使用github，可以参考SSH配置

###Node.js

一路下一步即可 Node.js下载

##安装hexo

#####安装
npm install hexo

#####查看版本
hexo version

#####执行下列命令创建你制定文件夹的文件
hexo init
cd
npm install //安装相关的依赖包

#####然后你就可以看到如下的指定文件夹
├── _config.yml //配置文件
├── package.json//应用程序的信息
├── scaffolds //模版文件夹
├── scripts //脚本文件夹
├── source//用户资源
| ├── _drafts
| └── _posts
└── themes//主题资源

##配置hexo
config的具体配置可以参考这里
config.yml配置

#####常用命令

hexo help #查看帮助
hexo init #初始化一个目录
hexo new "postName" #新建文章
hexo new page "pageName" #新建页面
hexo generate #生成网页，可以在 public 目录查看整个网站的文件
hexo server #本地预览，'Ctrl+C'关闭
hexo deploy #部署.deploy目录
hexo clean #清除缓存，**强烈建议每次执行命令前先清理缓存，每次部署前先删除 .deploy 文件夹**

#####主题
我使用的是yilia的主题，作者的博客是Hexo主题yilia

根据作者的博客应该很容易就可以搭建好这个主题

##编辑文章

新建文章

hexo new “new”

然后就会在source/_post文件夹出现一个新的new.md

使用markdown编辑器打开（我使用的是markdownPad）
打开new.md就会看到如下

title: 如何用hexo搭建博客并且绑定域名
date: 2015-09-04 10:12:21
tags: 
- hexo
- 博客
---
//正文

使用markdown语法编辑之后保存，使用

hexo-server

打开 localhost:0000 查看效果

##利用githubPages搭建博客

github会提供给每个用户300mb的空间，这对于搭建自己的博客是个再好不过的免费选择了

  • 创建一个仓库，名字为’你的用户名.io.github’
  • 使用hexo generate生成静态网页
  • 将”public”文件夹部署到github（我是用的是github for windows，很方便）
  • 直接访问”你的用户名.io.github”即可

##将博客绑定到你自己域名

我是用的是万网购买的的域名，29元第一年，并不是很贵

  • 在github的仓库中添加一个CNAME的文本文件，把你的域名添加进去（我的是huyifei.cn），效果如图
  • 如果成功了，就可以在Sittings的GitHub Pages里面找到Your site is published at http://huyifei.cn.
  • 然后在万网的域名解析里面添加cname即可（如图）
  • 万网要过一段时间等待DNS配置，大概过一段时间输入你的域名，就可以打开你的博客啦。
